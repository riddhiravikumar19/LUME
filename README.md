# ✨ Lume – Virtual Study Room

Illuminate your Study Arena!

Lume is a collaborative Virtual Study Room platform designed to help students stay productive, work together, and track their study progress effectively.

---

🚀 Features

- 🔐 User Authentication (Login/Register)
- 🏫 Study Room Creation & Management
- 📚 Personal & Public File Library
- ⏳ Pomodoro Timer (25 min study / 5 min break)
- ✅ To-Do List for task management
- 💬 Real-time Chat for collaboration
- 🏆 Leaderboard based on study activity

---

🛠️ Tech Stack

- Backend: Django (Python)
- Database: MongoDB
- Frontend: HTML, CSS, JavaScript
- Real-time: WebSockets (Django Channels)
- Deployment: Heroku / PythonAnywhere

---

⚙️ How to Run

1. Clone the repository
   git clone https://github.com/your-username/lume.git

2. Navigate to project folder
   cd lume

3. Create virtual environment
   python -m venv venv

4. Activate environment
   venv\Scripts\activate

5. Install dependencies
   pip install -r requirements.txt

6. Run the server
   python manage.py runserver
